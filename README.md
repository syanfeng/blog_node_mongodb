# Node+express+Mongodb 搭建博客Restful API

## 启动 MongoDB 服务器

window 平台使用 [Mongodb](http://www.runoob.com/mongodb/mongodb-window-install.html)

    C:\Program Files\MongoDB\Server\3.4\bin\mongod --dbpath C:\mongodb\data\db

## 启动 Node 服务
    npm install
    npm run start

## 访问服务
    http://localhost:8081/
